# Infoware-Assignment

### CRUD features in Node JS API  :

- Create Employee with multiple contact details (Relationship mapping)
- List Employee (with pagination)
- Update Employee
- Delete Employee
- Get Employee

### How to start the project

- git clone <repolink>

- npm i

- npm start
